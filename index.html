What is MTR Crafters?

Welcome to MTR Crafters! We are a community that aims to build the entire Hong Kong MTR (Mass Transit Railway) system in the Minecraft game using the MTR (Minecraft Transit Railway) Mod. ​With the MTR Mod, we are able to build the entire Hong Kong MTR System in Minecraft ​realistically!
